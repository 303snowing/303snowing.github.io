GitLab SpringBoot项目持续集成

2021年04月17日 658 字 大概 3 分钟

所有的环境都是基于docker的，我会给出涉及到的docker-compose.yml编排文件，使用docker-compose工具可以方便的拉起容器

可以使用vscode的remote ssh功能对服务器上的文件进行方便的远程编辑

服务器安装Docker环境

  1. 直接看官方文档就行：https://docs.docker.com/engine/install/

  2. 需要安装docker-compose编排工具：https://docs.docker.com/compose/install/

  3. 以上完成后，服务器应该能执行下面两条命令：

准备GitLab环境和gitlab-runner

  • GitLab

    在任意目录添加gitlab文件夹，进入gitlab文件夹新建docker-compose.yml，写入如下内容

    version: "3"

services:
  web:
    image: 'gitlab/gitlab-ce'
    container_name: 'gitlab'
    restart: always
    hostname: 'git.yourdomain'
    environment:
      GITLAB_OMNIBUS_CONFIG: |
        external_url "https://git.yourdomain"
        unicorn['port']=8081
        # 启用大文件支持
        gitlab_rails['lfs_enabled'] = true
        # ssh协议使用的访问地址和端口
        gitlab_rails['gitlab_ssh_host'] = 'git.yourdomain'
        gitlab_rails['gitlab_shell_ssh_port'] = 22
        # 邮件配置
        gitlab_rails['smtp_enable'] = true 
        gitlab_rails['smtp_address'] = "mail.yourdomain"
        gitlab_rails['smtp_port'] = 25
        gitlab_rails['smtp_user_name'] = "git"
        gitlab_rails['smtp_password'] = "your gitlab notify mail pass"
        gitlab_rails['smtp_domain'] = "mail.yourdomain"
        gitlab_rails['smtp_authentication'] = "login"
        gitlab_rails['smtp_enable_starttls_auto'] = true
        gitlab_rails['smtp_tls'] = true
        gitlab_rails['gitlab_email_enabled'] = true
        gitlab_rails['gitlab_email_from'] = "git@yourdomain"
        gitlab_rails['gitlab_email_display_name'] = "git@yourdomain"
        # 启用https支持
        nginx['enable'] = true
        nginx['client_max_body_size'] = '1024m'
        nginx['redirect_http_to_https'] = true
        nginx['redirect_http_to_https_port'] = 80
        nginx['redirect_http_to_https'] = true
        #nginx['redirect_http_to_https_port'] = 443
        nginx['ssl_certificate'] = "/var/ssl/yourdomain.crt"
        nginx['ssl_certificate_key'] = "/var/ssl/yourdoamin.key"
        nginx['ssl_ciphers'] = "ECDHE-RSA-AES256-GCM-SHA384:ECDHE-RSA-AES128-GCM-SHA256"
        nginx['ssl_prefer_server_ciphers'] = "on"
        nginx['ssl_protocols'] = "TLSv1.1 TLSv1.2 TLSv1.3"
        nginx['ssl_session_cache'] = "builtin:1000  shared:SSL:10m"
        nginx['listen_addresses'] = ["0.0.0.0"]
        nginx['http2_enabled'] = true

    ports:
      - 80:80
      - 443:443
    volumes:
      - /ext_data/docker_data/gitlab/ssl:/var/ssl
      - /ext_data/docker_data/gitlab/etc:/etc/gitlab:rw
      - /ext_data/docker_data/gitlab/logs:/var/log/gitlab:rw
      - /ext_data/docker_data/gitlab/data:/var/opt/gitlab:rw

    文件中需要修改的地方：

    • yourdomain是你自己的域名，没有的话直接填写gitlab服务器
    • yourdomain.crt和yourdomain.key分别是yu’my
